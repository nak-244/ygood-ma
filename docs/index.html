土地有効活用の案件・M&A案件の募集

ご案内

株式会社ワイグッドケアでは、新規開発の物件を常時募集しております。

土地有効活用案件や運営の受託などもお引き受けしております。
また、M&A案件についても検討しております。
地域性や規模に応じて適切な施設計画を立て失敗のない運営を行ってまいります。

対象運営形態

介護付有料老人ホーム

住宅型有料老人ホーム

サービス付き高齢者向け住宅

住宅型有料老人ホーム、サービス付き高齢者向け住宅の運営に関しては、訪問介護事業所と物件により、デイサービス事業所を併設します。

開発地域や規模によっては、新規事業の併設や元気な高齢者向けの高齢者マンション、さいたま市では介護施設と本社分室・研修施設との併設なども検討します。

対象エリア

東京都 23区（山手線管内・城西・城南）
埼玉県 さいたま市・草加市・八潮市・三郷市
神奈川県 横浜市・川崎市
千葉県 流山市
栃木県 宇都宮市・真岡市
茨城県 つくば市・守谷市

対象規模目安

建物

30～70室（1室18m2以上）

※東京23区や駅徒歩圏内であれば、延床999m2以下も検討可能ですが、地域によりデイサービス併設の場合があります。

※想定レンタブル比は50%～60%で検討しています。

敷地

200坪以上

※地域によりますが自動車通勤圏においてはスタッフの駐車場も必要です。

契約形態

20年～30年の一括借り上げ契約

※連帯保証人が必要な場合、当社は株式会社ワイグッドホールディングスの子会社ですので、親会社保証が条件となります。

※物件によっては契約期間の家賃保証や建築費保証をお断りすることがあることを予めご了承ください。

対象運営形態

チーム制の営業スタッフ

弊社営業スタッフは全担当チーム制となっております。そのため、担当営業が外出していて連絡が取れない、話がすすまないということはありません。採用は時間との戦いですので、スムーズな案件進捗を心がけております。
